## Troubleshooting basics

1. Check we are using the correct AWS REGION!!!!<br/>
  WARN: Sometimes the Web Console switch to the global region (IAM, ...) and then
  if fails to go back to our working region displaying no resources.

2. Check Data-center status for our regions
  <https://status.aws.amazon.com/govcloud>

3. Check all AWS resources in account:
   3.1. Go to Tag Editor: <https://console.aws.amazon.com/resource-groups/tag-editor/find-resources>
   3.2. Set "Regions" to "All Regions"
   3.3. Set "Resource Types" to "All supported resource types" 
   3.4. Click "Search Resources".

4. Check DISK USSAGE for EC2/ECS/K8s/... instances:<br/>
   It could be possible that some process is writting "too many logs".
