## Cloud Custodian: Multi-Cloud: Security, Governance, and Management

- License: Open Source, (CNCF Incubating Project under the Apache 2.0 license)

- <https://cloudcustodian.io/>
- <https://manheim-c7n-tools.readthedocs.io/en/latest/>


- Cloud Custodian enables users to be well managed in the cloud. 
  The simple YAML DSL allows you to easily define rules to enable a 
  well-managed cloud infrastructure, that's both secure and cost optimized.

-  It consolidates many of the ad-hoc scripts organizations have into a
   lightweight and flexible tool, with unified metrics and reporting.

- supports AWS, Azure, and GCP 
- Kubernetes, Tencent Cloud, and OpenStack support in beta.

- Real-time Compliance: actively enforce security policies by natively integrating
  with the cloud provider's control plane and remediating in real-time.

- Cost Management:
  - Setup off-hours to save money by turning off resources when 
    they're not being used. Garbage collect unused resources by looking 
    into utilization metrics. Easily tag and reap unused resources.

- Off Hours: 
  - Integrate Custodian with Terraform to ensure your infrastructure 
    comes up compliant right from the start, all defined in git for true 
    "Governance as Code" benefits. This feature is currently in Alpha.
