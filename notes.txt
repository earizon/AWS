# AWS Nomenclature [[{]]
* HIGH_AVAILABILITY: Service is respondent maybe in degraded mode                          [[{qa.ha]]
* FAULT-TOLERANT   : high_availability + compliant SLA
* AVAILABILITY ZONE: (AZ) Equivalent to a "local network". If two instances
   are in the same AZ, they can just put the MAC address of the target
   computer on it, and then the packet will get to the right place.
   It doesn't matter what IP address is on the packet!
* AWS REGION       : (Multi) Data center in some World geographical place with 2 or more
                       availabilities zones.                                                 [[}]]

# Troubleshooting basics [[{]]
* Check we are using the correct AWS REGION!!!!<br/>
  Sometimes the Web Console switch to the global region (IAM, ...) and then
  if fails to go back to our working region displaying no resources.
* Check Data-center status for our regions
  <https://status.aws.amazon.com/govcloud>

## Check DISK USSAGE for EC2/ECS/K8s/... instances:
* It could be possible that some process is writting "too many logs".

## Taxonomy 
* AWS Athena: Architect, design and implement the data migration
  solutions using AWS DMS, and should have good experience on
  integrating other components participating and data movement. Amazon
  Athena is an interactive query service that makes it easy to analyze
  data in Amazon S3 using standard SQL.
* AWS Aurora  An Architect, design and implement cloud-anchored data
  management and integration solutions using AWS Aurora Services in
  conjunction with other AWS Data services (On-prem on Cloud).
* AWS Backup  AWS Backup is a cost-effective, fully managed,
  policy-based service that simplifies data protection at scale.
* AWS BigData Design, develop, manage, implement and deploy Big Data
  business solutions and environment by managing and implement AWS
  Tools and Platforms such as Amazon Elastic MapReduce (EMR), Amazon
  Redshift, Amazon Kinesis and the rest of the AWS big data platform.
* AWS Billing Manage, administer and implement processes related to
  AWS Billing and Cost Management for an organization including
  services such as payment of AWS bills, monitor of usage, and budget
  of costs.
* AWS Cloud Control API An application programming interface (API) to
  manage the lifecycle of AWS and third-party services. Cloud Control
  API provides five operations for development to create, read, update,
  delete, and list cloud infrastructure.
* AWS Cloud9  A cloud-based integrated development environment (IDE)
  that writes, runs, and debugs codes with a browser. It includes a
  code editor, debugger, and terminal. Cloud9 comes prepackaged with
  tools for programming languages, including JavaScript, Python, and
  PHP (Hypertext Preprocessor).
* AWS CloudEndure This is a server replication service in AWS. It is
  used to replicate server image from on-prem to AWS or within AWS for
  server migration and DR protection. It is used as part of myNav
  migration process.
* AWS CloudFormation (deprecated in favor of AWS CDK)
* AWS CloudTrail  Service that enables governance, compliance,
  operational auditing, and risk auditing of AWS accounts.
* AWS CodeBuild Fully managed continuous integration service that
  compiles source code, runs tests, and produces software packages that
  are ready to deploy on AWS.
* AWS CodeCommit  Fully-managed source control service that hosts
  secure Git-based repositories on AWS.
* AWS CodeDeploy  Service that automates code deployments to Elastic
  Compute Cloud (EC2) and on-premises servers on AWS.
* AWS CodePipeline  Fully managed continuous delivery service for
  automating release pipelines on AWS.
* AWS CodeStar  Cloud-based service for creating, managing, and
  working with software development projects on AWS.
* AWS Command Line Interface (CLI)  It is a tool to manage AWS
  services. Control multiple AWS services from the command line and
  automate them through scripts.
* AWS Comprehend  A natural language processing (NLP) service that
  uses machine learning to find insights and relationships in text. It
  allows the use of AutoML capabilities in Amazon Comprehend to build a
  custom set of entities or text classification models that are
  tailored to an organization`s needs.
* AWS Config  Service that enables teams to assess, audit, and
  evaluate the configurations of their AWS resources.
* AWS Core Infrastructure Administer, manage, implement and deploy
  Amazon Web Service's core infrastructure for all its applications.
* AWS Database Migration Service  AWS Database Migration Service
  helps migrate databases to AWS quickly and securely. The source
  database remains fully operational during the migration, minimizing
  downtime to applications that rely on the database. The AWS Database
  Migration Service can migrate data to and from most widely used
  commercial and open-source databases.
* AWS DataSync  AWS DataSync is a storage replication service that
  simplifies, automates, and accelerates moving data between
  on-premises storage systems and AWS Storage services, as well as
  between AWS Storage services/ DR protection.
* AWS DBPaaS Migration  Optimize, refactor, re-architect and
  re-platform the On-Premise Databases and migrate them into AWS cloud
  by using Database Migration Assistant, AWS Schema Conversion tool,
  Double Take and Plate S pin tool.
* AWS Device Farm Enables to run test on multiple desktop browsers or
  real devices to speed up the execution of the test suite, and
  generates videos and logs to help identify issues with the app.
* AWS DevOps & Development  A set of flexible services designed to
  enable companies to more rapidly and reliably build and deliver
  products using AWS and DevOps practices. Allowing an organization to
  simplify provisioning and managing infrastructure, deploying
  application code, automating software release processes, and
  monitoring application and infrastructure performance.
* AWS Direct connect  AWS Direct Connect is a dedicated, highspeed
  networking service that provides an alternative to using the internet
  to connect to AWS. It is used to connect on-prem or client's other
  vendor location to AWS which reduces cost, increases bandwidth, and
  provides a more consistent network experience than internet-based
  connections. It is necessary service for Hybrid environment that
  requires highspeed band width.
* AWS Document DB Amazon DocumentDB is a fully managed proprietary
  NoSQL database service that supports document data structures and
  MongoDB workloads. It is a fast, scalable, highly available, and
  fully managed document database and is designed from the ground-up to
  give users the performance, scalability, and availability needed when
  operating mission-critical MongoDB workloads at scale.
* AWS DynamoDB  A fully managed proprietary NoSQL database service
  that is offered by Amazon.com that exposes a similar data model and
  derives its name from Dynamo, but has a different underlying
  implementation. Dynamo had a multi-master design requiring the client
  to resolve version conflicts and DynamoDB uses synchronous
  replication across multiple datacenters for high durability and
  availability.
* AWS EKS Anywhere  Amazon EKS Anywhere is a deployment option for
  Amazon EKS that allows customers to create and operate Kubernetes
  clusters on customer-managed infrastructure. It also provides
  on-premises Kubernetes operational tooling.
* AWS Elastic Beanstalk An orchestration service offered from Amazon
  Web Services for deploying infrastructure which orchestrates various
  AWS services, including EC2, S3, Simple Notification Service (SNS),
  CloudWatch, auto scaling, and Elastic Load Balancers.
* AWS Elastic Block Store (EBS) A software solution provides highly
  available, highly reliable block level storage volumes for use with
  Amazon Elastic Compute Cloud web service instances.
* AWS Elastic Disaster Recovery Allows monitoring replication and
  periodically performing non-disruptive recovery and failback drills.
  Launch recovery instances on AWS. Initiate data replication back to
  the primary site when the issue is resolved.
* AWS Elastic Kubernetes Service EKS  This is a managed Kubernetes
  service in AWS. Kubernetes is an open source system for automating
  the deployment, scaling, and management of containerized
  applications. It is used for application modernization and migrating
  legacy servers to cloud in addition to containerization and micro
  service without needing to install, operate, and maintain your own
  Kubernetes control plane or nodes.
* AWS ElastiCache A fully managed in-memory data store and cache
  service by Amazon Web Services. The service improves the performance
  of web applications by retrieving information from managed in-memory
  caches, instead of relying entirely on slower disk-based databases.
  It supports two open-source in-memory caching engines: Memcached and
  Redis.
* AWS Elemental AWS Elemental is a software that performs video
  encoding, decoding, transcoding, and pixel processing tasks on
  commodity hardware for adaptive bitrate streaming of video over IP
  networks.
* AWS Fargate AWS Fargate is a serverless compute engine that focuses
  on building applications without managing servers. AWS Fargate is
  compatible with both Amazon Elastic Container Service (ECS) and
  Amazon Elastic Kubernetes Service (EKS).
* AWS Glue  Architect, design and implement cloud-anchored query
  based analysis solutions using AWS Athena in conjunction with other
  AWS Data services (On-prem on Cloud). Integrated across a wide range
  of AWS service. AWS Glue natively supports data stored in Amazon
  Aurora and all other Amazon RDS engines.
* AWS GovCloud  AWS GovCloud regions have a set of security protocols
  and other features to host sensitive govt systems and data.
  Understand AWS GovCloud, host the environment, and help clients with
  government compliance requirements, including the International
  Traffic in Arms Regulations (ITAR) and Federal Risk and Authorization
  Management Program (FedRAMP).
* AWS GuardDuty Amazon Guard Duty is a threat detection service that
  continuously monitors for malicious activity & unauthorized behavior
  to protect AWS accounts, workloads, & data stored in Amazon S3.
  Analyzes continuous streams of meta-data generated from account &
  network activity found in AWS CloudTrail Events, Amazon VPC Flow
  Logs, and DNS Logs. Also uses integrated threat intelligence such as
  known malicious IP addresses, anomaly detection, and machine learning
  to identify threats more accurately.
* AWS High Availability Manage and implement AWS' HA platform, a
  system or component that is continuously operational for a desirably
  long length of time, to achieve business goals for an organization.
* AWS Internet of Things (IoT)  A managed cloud platform by Amazon
  that lets connected devices easily and securely interact with cloud
  applications and other devices.
* AWS Kinesis Amazon`s cloud-based service for managing and analyzing
  massive amounts of data from large, distributed data streams.
* AWS Lake Formation  Set up secure data lakes using a centralized,
  curated, and secure repository that stores all data, both in its
  original form and prepared for analysis.
  <https://www.infoq.com/news/2019/08/aws-lake-formation-ga/>
* AWS Lambda  An event-driven, serverless computing platform provided
  by Amazon as a part of the Amazon Web Services. It is a compute
  service that runs code in response to events and automatically
  manages the compute resources required by that code.
* AWS Lambda Administration A set of tasks that configure,
  troubleshoot, and manage AWS Lambda, is a compute service that
  automates code run without provisioning or managing servers.
* AWS Lex A service for building conversational interfaces into any
  application using voice and text. It provides the advanced deep
  learning functionalities of automatic speech recognition (ASR) for
  converting speech to text, and natural language understanding (NLU)
  to recognize the intent of the text.
* AWS Local Zones AWS Local Zones are a type of infrastructure
  deployment that places compute, storage, database, and other select
  AWS services close to populations and industrial centers.
* AWS Lookout Amazon Lookout for metrics uses machine learning to
  automatically detect and diagnose anomalies (i.e. outliers from the
  norm) in business and operational data, such as a sudden dip in sales
  revenue or customer acquisition rates.
* AWS Managed Blockchain  Amazon Managed Blockchain is a fully
  managed service that makes it easy to join public networks or create
  and manage scalable private networks using the open-source frameworks
  Hyperledger Fabric and Ethereum.
* AWS Management Console  Access and manage Amazon's growing suite of
  infrastructure Web services through the point-and-click, Web-based
  user interface of the Amazon Web Services Management Console which
  provides a quick, global picture of cloud computing environments.
* AWS Monitron  Amazon Monitron is an end-to-end system that uses
  machine learning to detect abnormal conditions in industrial
  equipment and enable predictive maintenance.
* AWS Neptune An Architect, design and implement cloud-anchored data
  management and integration solutions using AWS Neptune Graph DB
  Service in conjunction with other Data services (On-prem on Cloud).
* AWS Networking  A group of networking products that enables and
  organization to isolate its cloud infrastructure, scale its request
  handling capacity, and connect its physical network to its private
  virtual network.
* AWS Operations  Manage and administer operational business
  processes by implementing AWS platforms and tools.
* AWS OpsWorks  AWS Configuration management service that provides
  managed instances of Chef and Puppet.
* AWS Outposts  Design and deliver AWS services in client`s
  datacenters as a hybrid cloud platform, enabling new scenarios for
  your modern applications, such as edge and disconnected environments,
  or meet specific security and compliance requirements.
* AWS Panorama  AWS Panorama is a machine learning appliance and
  software development kit (SDK) that brings CV to on-premises internet
  protocol (IP) cameras.
* AWS Polly A cloud service by Amazon Web Services, a subsidiary of
  Amazon.com, that converts text into lifelike speech. It allows to
  create applications that talk, and build entirely new categories of
  speech-enabled products.
* AWS Proton  An automated infrastructure as a code and two-pronged
  automation framework for provisioning and deployment of serverless
  and container-based applications. Creation of versioned service
  templates that define the standardized infrastructure and deployment
  tooling.
* AWS Quantum Ledger Database (QLDB)  Amazon Quantum Ledger Database
  (QLDB) is a fully managed ledger database that provides a
  transparent, immutable, and cryptographically verifiable transaction
  log. It is a non-relational database as a service (DBaaS) to maintain
  a verifiable history of data changes in an application.
* AWS Redshift  A petabyte-scale data warehouse service, designed for
  analytic workloads and connects to standard SQL-based clients and
  business intelligence tools.
* AWS Rekognition An image analysis service available in the Amazon
  AI suite. Through the Amazon Rekognition API, enterprises can enable
  their applications to detect and analyze scenes, objects, faces and
  other items within images.
* AWS Relational Database Service (RDS) Implementation of data base
  system on cloud, automating provisioning and administration tasks,
  with high efficiency. It provides cost-efficient and resizable
  capacity while automating time-consuming administration tasks such as
  hardware provisioning, database setup, patching and backups.
* AWS Robomaker A cloud solution for robotic development to simulate,
  test and securely deploy robotic applications at scale. Understand
  how to leverage AWS Robomaker to develop client applications that can
  be used to provide value.
* AWS S3 (Simple Storage Service) An online storage web service that
  provides unlimited storage for an organization through a simple Web
  services interface.
* AWS Schema Conversion Tool  The AWS Schema Conversion Tool (AWS
  SCT) is a standalone application that provides a project-based user
  interface. AWS SCT is available for Fedora Linux, macOS, Microsoft
  Windows, and Ubuntu Linux, . AWS SCT is supported only on 64-bit
  operating systems.
* AWS Security Hub  AWS Security Hub is a framework service in
  Preview that gives you a comprehensive view of your high-priority
  security alerts and compliance status across AWS accounts. It reduces
  the effort of collecting and prioritizing security findings across
  accounts, from AWS services, and AWS partner tools.
* AWS Simple Notification Services  A notification service provided
  as part of Amazon Web Services. It provides a low-cost infrastructure
  for the mass delivery of messages, predominantly to mobile users.
* AWS SimpleDB  A distributed database used as a Web service in
  concert with Amazon Elastic Compute Cloud (EC2) and Amazon Simple
  Storage Service (S3).
* AWS Snow Family Snow Family comprises AWS Snowcone, AWS Snowball,
  and AWS Snowmobile, which offers a number of physical devices and
  capacity points. These services help physically transport up to
  exabytes of data into and out of AWS. These services integrate with
  AWS security, monitoring, storage management, and computing
  capabilities.
* AWS Step Functions  A visual workflow service that uses AWS
  services to build distributed applications, automate processes,
  orchestrate microservices, and create data and machine learning (ML)
  pipelines.
* AWS Systems Manager A collection of capabilities for configuring
  and managing Amazon EC2 instances, on-premises servers and virtual
  machines, and other AWS resources at scale.
* AWS Transcribe  An automatic speech recognition (ASR) service that
  makes it easy for developers to add speech-to-text capability to
  their applications.Amazon Transcribe API, helps analyze audio files
  stored in Amazon S3 and have the service return a text file of the
  transcribed speech.
* AWS Transfer Family AWS Transfer Family is a secure transfer
  service that enables the transfer of files into and out of AWS
  storage services.
* AWS Translate A neural machine translation service that delivers
  fast, high-quality, and affordable language translation.
* AWS VPN AWS Virtual Private Network is a network service which
  establishes secure connections between on-premises networks, remote
  offices, client devices, and the AWS global network. AWS VPN is
  comprised of two services: AWS Site-to-Site VPN and AWS Client VPN.
  Each service provides a highly-available, managed, and elastic cloud
  VPN solution to protect network traffic. It is necessary service for
  Hybrid environment support and works as a backup for Direct connect.
* AWS Wavelength  AWS Wavelength embeds AWS compute and storage
  services within 5G networks, providing mobile edge computing
  infrastructure for developing, deploying, and scaling
  ultra-low-latency applications.
* AWS X-Ray Helps developers analyze and debug production,
  distributed applications, such as those built using a microservices
  architecture.
